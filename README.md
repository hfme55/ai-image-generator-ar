
# أداة مُولد الصور بالذكاء الاصطناعي (واجهة عربية)

هذه نسخة جاهزة لموقع بسيط يُنشئ صورًا بالذكاء الاصطناعي من وصف نصي (Prompt).  
المشروع مُعد للنشر السريع على استضافة مثل **Vercel** أو أي سيرفر ويب بسيط.

## ملاحظات مهمة قبل التشغيل
1. **توكن Hugging Face**: افتح الملف `index.html` وابحث عن السطر:
   ```js
   const HUGGING_FACE_TOKEN = "YOUR_HF_TOKEN";
   ```
   واستبدل `"YOUR_HF_TOKEN"` بالتوكن الحقيقي من: https://huggingface.co/settings/tokens
2. **كود الإعلان**: ضع كود جافا سكربت/HTML لشبكة الإعلانات داخل `<div id="top-ad" class="ad">` في `index.html`.
3. بعض نماذج Hugging Face قد تقيّد عدد الطلبات أو تحتاج إلى خطة مدفوعة. تأكد من شروط استخدام النموذج والحدود.
4. هذا المشروع مصمم كنقطة انطلاق — يمكنك تخصيصه من ناحية التصميم، إضافة نظام مستخدمين، أو صف انتظار (Queue) إذا زاد الضغط.

## بنية الملفات
- `index.html` — الواجهة الرئيسية (عربية)
- `styles.css` — أنماط CSS
- `README.md` — هذا الملف

## نشر على Vercel (سريع)
1. أنشئ repo على GitHub وضع الملفات فيه.
2. ادخل Vercel → New Project → Import Git Repository.
3. املأ متغير البيئة `HUGGING_FACE_TOKEN` إذا أردت إخفاءه بدل وضعه مباشرة في الملف.

## ملاحظات تقنية
- طريقة الاتصال هنا تعتمد على **Hugging Face Inference API** — وهي طريقة سريعة وسهلة للمشروعات الصغيرة.
- إن أردت دعم عدد كبير من المستخدمين **بلا حدود**، ستحتاج إلى استضافة نموذج محلي على GPU أو حلول مدفوعة لتوفير أداء مستمر.
